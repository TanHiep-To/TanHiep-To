# Content Structure

This folder is prepared for a future data-driven site build. It keeps structured data separate from the HTML pages so the portfolio can scale cleanly.

## Files
- `projects.json`: project catalog for the portfolio.
- `blog.json`: blog metadata for the writing section.
- `experience.json`: work and education timeline data.

The current site renders static HTML, but these JSON files can be used by future scripts or a static-site generator.
